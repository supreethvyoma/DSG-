This archive contains the full React + Express project for manual Node.js deployment on Hostinger.

Recommended steps:
1. Upload and extract this zip.
2. Set backend/.env values.
3. Run npm install.
4. Run npm run build.
5. Start with npm start.

If Hostinger's framework wizard still rejects this archive, create the Node app first and upload/extract this archive through File Manager instead of the framework importer.
